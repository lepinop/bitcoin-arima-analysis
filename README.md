# bitcoin-arima-analysis
Modelar y predecir el precio de cierre de Bitcoin (BTC-USD) utilizando modelos ARIMA con diferentes criterios de selección (AIC, BIC, HQIC) y diferentes horizontes de predicción (7, 14, 21 y 28 días), comparando métodos rolling y estático.
